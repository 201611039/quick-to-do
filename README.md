<p> How To Install </p>

    1. git clone https://github.com/201611039/quick-to-do.
    2. cd quick-to-do.
    3. composer install.
    4. cp .env.example .env.
    5. Set up database name and create database.
    6. php artisan key:generate.
    7. php artisan migrate.
    8. php artisan serve.
    
<p> Well! thats it... </p>
